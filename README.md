# CWA 衛星雲圖／雷達回波圖 網址產生器

一鍵產生中央氣象署(CWA)衛星雲圖與雷達回波圖的下載網址。

## 功能

- **衛星雲圖**：支援臺灣、東亞、全景三種區域（可見光、紅外線、真實色等）
- **雷達回波圖**：支援臺灣全域/區域，含/不含地形，大圖/小圖
- 自動帶入當天日期，起始/結束時間智慧預設
- 一鍵產生所有網址
- 點擊網址即可預覽圖片
- 自動播放動畫效果
- 滾輪縮放圖片

## 使用方式

直接開啟 [GitHub Pages](https://xun73.github.io/cwa_image/) 即可使用。

或下載 `index.html` 用瀏覽器開啟。

## 資料來源
